pasos generales para inciar a trabajar con git


1.- colocarnos en la carpeta que vamos a trabajar.
    Ya sea en la consola de Git Bash o en nuestro editor 

2.- ejecutar el comando "git init" y listo

3.- ejecutamos el comando "git status", verificando que todo esta listo
    para iniciar a trabajar.
    

    acontinuacion se revisa el barrado de archivos y la restauracion de archivos.

    bueno primero se borra el archivo, despues se informa a git que se ha
    borrado pasándolo al "staged" y despues se hace el "commit" para borrarlo 
    en donde estaba haciendo el respaldo.
    
    ya hemos visto varios comando, como los de restaurar "git restore [Name_file]"
    o tambien "git --restore [Name_file]"

    tambien se ha visto elcomando "git log", para ver el historial de lo que se ha realizado
    asi como tambien en una forma mas simplificada el comando "git log --oneline"

    por cierto para observar el estado de como se van realizando tareas tambien se puede 

    usar el comando "status -s" el cual nos presenta de manera mas concisa la información de
    lo que se ha realizado en el proceso
